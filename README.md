# 📊 Dashboard de Vendas no Excel

## 📌 Sobre o Projeto

Este projeto consiste na criação de um dashboard de vendas utilizando Microsoft Excel, com foco em organização, tratamento e visualização de dados para análise de indicadores de desempenho.

O dashboard permite acompanhar métricas importantes de vendas de forma visual, facilitando a tomada de decisões baseada em dados.

---

## 🚀 Tecnologias Utilizadas

- Microsoft Excel
- Tabelas Dinâmicas
- Gráficos Dinâmicos
- Segmentação de Dados
- Power Query

---

## 📈 Indicadores Apresentados

- Faturamento Total
- Quantidade de Vendas
- Ticket Médio
- Produtos Mais Vendidos
- Ranking de Vendedores
- Evolução das Vendas
